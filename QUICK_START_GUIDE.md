## 💰 Bank Details Usage Instructions

### How Customers See It:
1. Click "💚 Donate" in navigation → Goes to Bank Details section
2. Choose payment method (bank transfer, UPI, or international)
3. Click "Copy" button to copy account number/details to clipboard
4. Paste in their banking app or website
5. No need to call the orphanage for details!

### How to Update Details:
1. Open the HTML file in a text editor
2. Find the bank details section (marked with `<!-- BANK DETAILS -->`)
3. Replace the sample values with your actual details:
   - Account Number
   - IFSC Code
   - UPI ID
   - Phone Number
   - SWIFT Code
   - Contact Number
4. Save and test

**No technical skills needed** - Just simple find & replace!

## 🆘 Troubleshooting

### Images Not Showing?
- Check image URLs are complete (http://... or https://...)
- Verify image files exist on server
- Check file permissions
- Clear browser cache

### Copy Buttons Not Working?
- Ensure modern browser (Chrome, Firefox, Safari, Edge)
- Check JavaScript is enabled
- Clear cache and refresh page
- Check browser console for errors (F12)

### Layout Issues on Mobile?
- Clear browser cache
- Try different mobile browser
- Check viewport meta tag exists
- Ensure CSS loads correctly

### Bank Details Not Visible?
- Check HTML syntax is correct
- Verify section ID is `bank-details`
- Check CSS is loading
- Look for any JavaScript errors
